LFinance - Geração do EXE

1. Abra o PowerShell ou CMD na pasta do projeto.
2. Execute:
   gerar_exe.bat

O executável será criado em:
   dist\LFinance.exe

Observações:
- O banco de dados NÃO fica dentro da pasta do programa.
- O banco fica em %LOCALAPPDATA%\LFinance\lfinance.db.
- Isso protege os dados em futuras atualizações do EXE.
- Os assets e o ícone já estão preparados para funcionar no executável.
